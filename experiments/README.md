# Cathedral Real Experiment Research

This directory contains research experiments, data collection, and creative explorations for the Cathedral Real project.

## Structure

Each experiment should be contained in its own subdirectory with the following structure:

- `README.md`: Hypothesis, goals, and methodology.
- `data/`: Collected data (if any).
- `src/`: Source code specific to the experiment.
- `results/`: Analysis and conclusions.

## How to Start a New Experiment

1. Create a new directory: `mkdir experiments/my-new-experiment`
2. Copy the template: `cp experiments/TEMPLATE.md experiments/my-new-experiment/README.md`
3. Define your goals and start coding!
