- Each card = one alchemical operation + one creative archetype
- Crowley's Thoth attributions as one layer
- Case's B.O.T.A. colors as another
- Fortune's pathworking descriptions as navigation

**Minor Arcana (56):**

- Wands (Fire) = Will, inspiration, initiation
- Cups (Water) = Emotion, intuition, reception
- Swords (Air) = Intellect, analysis, communication
- Disks (Earth) = Manifestation, craft, completion

**Court Cards (16):**
- Pages = Students of the element
- Knights = Active practitioners
- Queens = Masters of reception
- Kings = Masters of projection
